Coding Tasks

Welcome in Coding Task page !

To create your first task, click the plus icon above the line.

Create your first mini task !

After entering and confirming the task name, you should see three labels named IDEAS, TESTING, RELEASED.

In each label section you can create a new task, which you can drag to another section if you want to change the status of your mini task.

Edit, add to favorite and delete !

You can also edit the task name by double-clicking the task above the line, add tasks to your favorites by clicking the star icon and delete a task by clicking the trash icon next to the star.
